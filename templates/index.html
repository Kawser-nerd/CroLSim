Please upload your tools here to train and detect software similarity. 
You can upload software applications in three ways.
1. You can select the source folder of a software application. It will automatically upload all the source codes of the software application to the server.
2. You can select multiple software applications at a time. The system will upload source codes of all the software applications automatically by preserving their uniqueness and originality
3. You can select a folder of having all the software applications together. This system will automatically identify the software application and will upload it to the system keeping their originality and uniqueness.

To search cross language similar software applications of a specific software application, you first need to upload the software application. After completion of uploading the software application, similar software applications from the repository will be popped up on the designated area.

To search with software application name, an user needs to put the name in the search section. Once the user put the name, it will send the name to the backend server. And the backend server will execute the query, try to match it with the software applications name in the repository and result it with the best match software applications. Through this way, not only the software application with very related name will popped up, but also the software applications with functional similar source codes will be listed.
